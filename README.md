# LearnUp English App

Minimal Duolingo-style English learning mobile app.

## MVP Scope

This first version is intentionally simple:

- English-only learning content
- No AI feedback
- No backend
- No admin panel
- Local JSON content
- Local XP and streak tracking
- Expo + React Native + TypeScript
